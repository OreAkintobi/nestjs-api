## Description

This is a simple crud api built with [Nest](https://github.com/nestjs/nest) with a [Prisma](https://prisma.io/) relational database and TypeScript.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev
```

## Test

```bash
# run test server
$ yarn db:test:restart

# e2e tests
$ yarn test:e2e
```
